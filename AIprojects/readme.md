# AI projects
## Intro 
This folder is collection of various python code that I have asked various AI models to generate. 

## Files
[countdown.py](countdown.py)  gives back how many years, months, and days until a future date. Generated by Grok
[plotter.py](plotter.py) plots a graph of 3 columns of data from a csv file. Generated by Grok
[pressureFlowCalcGPT5m.py](pressureFlowCalcGPT5m.py) calculates pressure and flow rates for piping systems with inputs for various fittings. Generated by GPT-5 mini GitHub and Zed.dev editor.
